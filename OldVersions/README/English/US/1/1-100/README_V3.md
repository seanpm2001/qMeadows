# Description

qMeadows is the project for my dream operating system that runs on Quantum computers. I have been developing this idea since 2013. You will want to read more about it [here.](https://gist.github.com/seanpm2001/f1970d7a745fb4f8acde0e9364d727a1)

***

## About

See above. qMeadows is my main, dream operating system project that runs on Quantum Computers. Idea development started in 2013, but programming work only started circa 2016/2017/2018. It is far from complete, and it has lots to it. I hope to complete this operating system to my design in my lifetime, it is one of my main life goals.

The current major challenge preceding full development is difficulty in getting my visions properly translated into the project interface.

Who Meadows is intended for:

- Quantum Computer users
- Users who want as must customization as possible
- Users who don't like minimalism

You will want to see documentation repositories, instead of this README. Information hasn't properly been consolidated here yet.
